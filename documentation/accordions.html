Accordions are modeled after Bootstrap accordions, with modifications for accessibility (larger hit-state, open/closed icons, and right-to-left support).

For screan readers, role="heading" and aria-level="X" attributes are present on the span tag with class="title". In this way, we can avoid nesting headers within links (which can have a negative impact on accessibility).

In the code example below, Bootstrap's header class h4 is applied to the link tag, while the span with class="title" has role="heading" and aria-level="3" attributes (for screen readers).
