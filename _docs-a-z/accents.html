Gradient Veneer

Apply a directional gradient to any selector using the .gradient-veneer-{x} utilities. Where x specifies the gradient's direction: top, right, bottom, or left (Figuire 1 below).

Gradient's are a transparent-to-opaque single color. They inherit their color from the nearest ancestrial background color (usually applied with theme colors). For a gradient that is transparent at the top, and opaque at the bottom, use .gradient-veneer-bottom.

Consider the following visual representing our four directional gradients. Each with a different ancestrial theme applied.

Gradient veneer utilities place the gradient (as a psuedo element) above all child selectors on the z index. With the above examples, we applied our gradient veneer utility to the .backdrop-image selector. This allows the gradient to appear above its child <img> tag (Figure 2, lines 3-5).

Subtle Fill

The .subtle-fill utility class is perfect for adding a light background tint to any element. Subtle fills inherit their color from the text color of the element on which they are placed.
