Gradient Veneer

Apply a directional gradient to any element using the .gradient-veneer-{x} utilities. Where x specifies the gradient's direction (top, right, bottom, left).

For example, use .gradient-veneer-bottom for a gradient that is transparent at the top, and opaque at the bottom. Gradient's are single-color. Their color is inherited from the theme color applied to its nearest ancestor.
