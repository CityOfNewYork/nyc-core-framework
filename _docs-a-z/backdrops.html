--- layout: documentation title: "Backdrops" ---

Backdrops provide a useful design pattern—placing a translucent image behind contextual content. Backdrops offer two distinct style variants: Image proportional and fixed height.

{% capture as_body %}

Backdrops remove the need for CSS background images. Feature images exist within the HTML, so screen readers can read the text of the image's alt attribute.

Breakpoint Modifiers

When using image size proportional backdrops for featured content (like the full-width example below) the content may become too tall for the image on small viewports.

Include a stack breakpoint modifier (e.g. .backdrop--stack--md) to the .backdrop selector and the backdrop's image is placed above the content vertically until the viewport reaches the desired breakpoint specified in the modifier.
