Grid Cell Widths

There are several ways to apply widths to grid cells. To apply the same width to all .grid__cell selectors, apply the .grid--cells-{x} modifier class to the .grid selector, where x equals their desired width (From 1-12).

For example, to make a three-column grid using the 12 column system, apply .grid--cells-4 to .grid (12 total columns divided by 3 columns equals 4).

Specific widths can be applied to individual columns. On any .grid__cell selector, apply the modifier class .grid__cell-{x} where x equals the desired width.
