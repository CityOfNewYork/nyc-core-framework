Quickly prototype and build responsive layouts with a simple (and likely familiar) flexbox 12 column grid system.

Grid Selectors

The Grid component requires a parent .grid selector, and at least one instance of a child .grid__cell selector.

The .grid__cell selectors inherit the width of their parent .grid and stack on top of one another when a width modifier class is not present. If a width modifier specific to a breakpoint is present, .grid__cell selectors remain stacked until the viewport width reaches that breakpoint.

Grid Cell Width

To specify width to grid cells, use the .grid--cell-{x} modifier class, where x equals their desired width (a number from 1 to 12).

For example, to make a three-column grid using the 12 column grid system, apply .grid--cell-4 directly to the .grid selector (Figure 2, line 1).

The .grid--cell-{x} modifier class works on individual .grid__cell selectors, too.

Breakpoint Modifiers

To create a responsive layout, chain a breakpoint modifier (e.g. .grid--cell-{x}--{y} where x again equals the cell width, and y specifies th desired breakpoint.

For example, to make a three-column layout at the medium breakpoint, apply .grid--cell-4--md to the .grid selector (Figure 4, line 1).
